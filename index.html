เชื่อมโยง Google Sheets ผ่าน Google Forms (ไร้สคริปต์)

บันทึกข้อมูลแบบเรียลไทม์ผ่าน Google Forms และอ่านข้อมูลออนไลน์กลับมาแดชบอร์ดโดยใช้ลิงก์เผยแพร่เว็บ (TSV):

1. ฟอร์มสำหรับสแกนบันทึกเข้าร่วม (Scan Log Form)

2. ฟอร์มสำหรับจัดการกิจกรรม (Activity Form)

เนื่องจากข้อมูลทั้งหมดถูกจัดเก็บอยู่บนเครื่องที่ใช้สแกนนี้ (LocalStorage) หากคุณเปิดสแกนหลายเครื่องแยกกัน คุณสามารถรวบรวมข้อมูลทั้งหมดเข้าด้วยกันได้ตามขั้นตอนดังนี้:

  1. ที่เครื่องลงทะเบียน/เครื่องย่อย ให้กดปุ่ม "ส่งออกข้อมูลสำรอง (JSON)" ด้านล่างเพื่อดาวน์โหลดไฟล์สำรอง
  2. รวบรวมไฟล์สำรองส่งไปที่เครื่องหลัก (เช่น ผ่านทาง LINE, อีเมล หรือ Flash drive)
  3. เปิดเว็บไซต์นี้ในคอมพิวเตอร์หลัก ไปที่หน้าการตั้งค่านี้ แล้วกดเลือกไฟล์ JSON ที่ได้รับเพื่อ "นำเข้าและควบรวมข้อมูล"
  4. ระบบจะรวมข้อมูลประวัติการสแกนและสถิติต่างๆ ให้อัตโนมัติโดยไม่ทำข้อมูลเดิมสูญหาย

1. ส่งออกข้อมูลสำรองจากเครื่องนี้

2. นำเข้าไฟล์สำรองเพื่อรวมร่างข้อมูล

ล้างระบบเพื่อเริ่มต้นใหม่

คู่มือการเผยแพร่ระบบผ่าน GitHub Pages

หากคุณต้องการอัปโหลดโปรเจ็กต์นี้ขึ้นบน GitHub เพื่อให้สามารถเปิดใช้งานผ่านมือถือ แท็บเล็ต หรือเครื่องอื่นๆ ผ่านลิงก์อินเทอร์เน็ตได้ ให้ทำตามขั้นตอนดังนี้:

ขั้นตอนการอัปโหลดขึ้น GitHub:

  1. เปิดคอมพิวเตอร์ของคุณที่โฟลเดอร์นี้
  2. ใช้คำสั่ง Git เพื่อ Commit และ Push โค้ดทั้งหมดขึ้นไปยังคลังโค้ดของคุณ
  3. หรือหากต้องการอัปโหลดทางหน้าเว็บ: ให้ไปที่หน้าเว็บ GitHub Repository eduAct ของคุณ
  4. ลากไฟล์ index.html, style.css และ app.js ไปวางบนเว็บไซต์ GitHub เพื่ออัปโหลดขึ้นหน้าเว็บ

ขั้นตอนการเปิดใช้งาน GitHub Pages:

  1. ในหน้า GitHub Repository ไปที่แท็บ Settings (การตั้งค่า)
  2. เลือกเมนู Pages ด้านซ้ายมือ
  3. ที่หัวข้อ Build and deployment ใต้ Source ให้เลือก Deploy from a branch
  4. ที่หัวข้อ Branch ให้เลือก main (หรือ master) และเลือกโฟลเดอร์ / (root) จากนั้นกด Save
  5. รอประมาณ 1-2 นาที คุณจะได้ลิงก์ของเว็บไซต์ เช่น https://tswiwat.github.io/eduAct/ สำหรับแจกจ่ายและเปิดใช้งานกล้องลงทะเบียนได้ทันที!
ข้อแนะนำเรื่องสิทธิ์เข้าถึงกล้อง:
เบราว์เซอร์บนมือถือจะอนุญาตให้ใช้กล้องได้เฉพาะเว็บไซต์ที่เป็น HTTPS ที่ปลอดภัยเท่านั้น ลิงก์ที่สร้างโดย GitHub Pages (HTTPS) จะพร้อมให้เข้าใช้งานกล้องบนโทรศัพท์มือถือระบบ iOS และ Android ได้ทันทีอย่างไม่มีปัญหา
